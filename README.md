# dotfiles

Configuration files repository.

## Installation

1. Clone the repository to HOME directory.
   ```
   git clone git@github.com:mmateja/dotfiles.git $HOME/.dotfiles
   ```

1. Run setup script and follow instructions.
   ```
   ./setup.rb
   ```

## Notes
